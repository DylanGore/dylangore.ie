My personal website and portfolio - 2021 edition!

## Requirements

- Hugo (Extended Edition)
- Node JS (v14)

## Development

Start development server with live reload:

```bash
hugo server
```

Using Node:

```bash
npm run dev
```

## Build

```bash
hugo --minify
```
